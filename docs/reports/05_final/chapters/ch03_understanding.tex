\subsection{Dataset Description}
\tobewritten{Anas El Kartouti}

The CICIDS2017 dataset, developed by the Canadian Institute for Cybersecurity, provides labeled network flow data generated in a controlled environment replicating realistic traffic conditions. The specific file used is \texttt{Friday-WorkingHours-Afternoon-PortScan.pcap\_ISCX.csv}, containing 286,467 flows with 79 raw features.

\centering

\midrule

Number of raw features & 79 \\
PortScan class & 158,930 (55.48\%) \\
