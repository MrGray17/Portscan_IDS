\section*{Abstract}
\addcontentsline{toc}{section}{Abstract}

In the context of increasing network reconnaissance threats, this report presents the design, implementation, and validation of AEGIS Entropy --- an AI-based Network Intrusion Detection and Prevention System specialized in detecting and responding to Port Scan attacks. The system combines supervised Machine Learning classifiers (Random Forest, XGBoost) with an unsupervised anomaly detector (Isolation Forest) to identify port scanning activity from network flow features. Beyond detection, AEGIS integrates a Moving Target Defense (MTD) layer that actively reshapes the network attack surface through port rotation, OS fingerprint spoofing, honeypot deployment, and automated IP blocking. The complete system operates in two modes: an offline mode for analysis of pre-captured Nmap scans, and a live mode for real-time detection and response on a controlled LAN environment. Experimental results on the CICIDS2017 dataset demonstrate detection accuracy of 99.91\% with a False Positive Rate of 0.11\%, exceeding all project success criteria. The defense subsystem achieves full-stack activation in under 520ms, with attacker reconnaissance time increased by 420\% under MTD.
